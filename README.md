What Medora does:

• Detects dangerous medication interactions using clinical drug-interaction databases

• Verifies medication combinations instantly

• Uses OCR to scan prescriptions and automatically extract prescribed medications

• Creates a patient profile with medication history for continuous monitoring and safer follow-up

• Generates AI-powered explanations in Algerian Darija

• Provides a dual interface for both patients and pharmacists

• Includes an AI chatbot assistant for pharmacists to support clinical decision-making

• Adapts clinical verification to the Algerian pharmaceutical context
